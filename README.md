# PDF-RAG-Chatbot
A Retrieval-Augmented Generation (RAG) chatbot that lets you upload any PDF and ask questions about it in natural language. Uses LangChain, ChromaDB, and HuggingFace embeddings to retrieve relevant context, then Groq's Llama 3.1 to generate grounded, source-cited answers — reducing hallucination compared to a plain LLM chatbot.
